# Project 1: ETL Pipeline

A professional ETL pipeline built with Python and Pandas.

## Overview
This project demonstrates a real-world ETL (Extract, Transform, Load) pipeline
using a Kaggle dataset, built as part of a data engineering portfolio.

## Tech Stack
- Python
- Pandas
- Kaggle API

## Project Structure
- data/raw/ — original source data (never modified)
- data/processed/ — cleaned and transformed output
- src/ — pipeline scripts
- notebooks/ — exploratory analysis
